## 1. Jenkins 배포 실패 — `cannot find symbol: class VoUtil`

`CashFlowCmptContoroller.java:14` 에서 `kobc.core.util.VoUtil` import 못 찾음.

**원인:** import 경로 오류. 실제 `VoUtil` 위치는 다른 패키지(`kobc.cmmn.util` 등).
**진단:** Eclipse `Ctrl+H` → File Search → `class VoUtil` 검색.

(파일명도 오타: `Contoroller` → `Controller`. 빌드 에러 원인은 아니지만 추후 수정.)

## 2. 그리드 컬럼 공통코드 매핑 누락

분납주기/통화코드 컬럼이 코드값(`1083`, `KRW`) 그대로 표시됨.

**원인:** `setCommonCode` 매핑이 셀렉트박스(`sbx_*`)에만 적용. 그리드 컬럼에는 별도 매핑 필요.

```js
var codeOptions = [
    { grpCd : "5810", compID : "sbx_stacTaskClssCd" },
    { grpCd : "1083", compID : "sbx_grnfPayPrdClssCd" },
    { grpCd : "1083", compID : "grd_IdstAmtCmpt:grnfPayPrdClssCd" },   // 추가
    { grpCd : gcm.CODE_COMBO_ID.CURR, compID : "sbx_dprcAmtCurCd" },
    { grpCd : gcm.CODE_COMBO_ID.CURR, compID : "grd_IdstAmtCmpt:dprcAmtCurCd" } // 추가
];
```

> 그리드 컬럼 매핑 시 `compID:"그리드ID:컬럼ID"` 형식 사용.

## 3. 엑셀 다운로드 — 활성 탭 그리드만 다운로드

### 시행착오

| 시도 | 결과 |
|------|------|
| `grd_IdstAmtCmpt.isVisible()` | TypeError. 그리드에 `isVisible()` 없음 |
| `grd_A.getRowCount() > 0 ? A : B` | 두 탭 다 데이터 있으면 무조건 A로 감 |
| `getSelectedIndex()` | WebSquare 탭에 미지원 |
| `getSelectedTabIndex()` | **정상 동작** |

### 최종 코드

```js
scwin.btn_excelDownloadGrcd_onclick = function(e) {
    var idx = teb.getSelectedTabIndex();
    var targetGrid = idx == 0 ? grd_IdstAmtCmpt : grd_B;
    
    if (targetGrid.getRowCount() == 0) {
        com.win.message("IZZ0048", "데이터|먼저 조회를 하세요.");
        return;
    }
    var option = { autoSizeColumn : true, useDataFormat : true };
    option.fileName = "이차보전금액검증_" + com.date.getServerDateTime("yyyyMMddHHmmss") + ".xlsx";
    com.data.downloadGridViewExcel(targetGrid, option);
};
```

> WebSquare 탭에서 활성 탭 인덱스: `[탭컨트롤ID].getSelectedTabIndex()`. **개별 탭 ID(`tabs2`/`tabs3`) 말고 부모 탭컨트롤 ID(`teb`) 사용.**

## 4. 그리드 세로 스크롤 2개 표시

```xml
<w2:content style="height:654px;">          <!-- 부모 654 -->
    <w2:gridView style="height: 725px;">    <!-- 자식 725 (부모보다 큼) -->
```

**원인:** 부모보다 자식이 크면 부모에서 스크롤 1개, 그리드 자체에서 스크롤 1개 = 2개.

**해결:** 그리드 height를 `654px` 이하 또는 `height:100%` 로 변경.
