## 처리한 이슈

### 1. 수정사유 alert 순서 문제

- **증상:** 수정 버튼 클릭 시 수정사유 검증 alert보다 "저장하시겠습니까?" alert이 먼저 뜸
- **원인:** `com.win.message`는 비동기 — if 블록이 끝나기 전에 다음 줄 실행됨. `return` 누락.
- **해결:**
```js
if (com.util.isEmpty(mdfcnRsnCn)) {
    com.win.message("EZZ00089", "수정사유");
    return; // 필수 — 없으면 다음 줄 즉시 실행됨
}
com.win.message("QZZ0001", null, function() {
    // 저장 로직
});
```
- **원칙:** WebSquare `com.win.message`는 비동기. 검증 후 흐름 차단이 필요하면 반드시 `return`.

### 2. 그리드 만기일 경과 시 자동 "종료" 표시

- **기능:** 조회 후 만기일(`mtryYmd`) 경과 건 자동으로 상태 "종료" 표시
- **버그:** 조건문 `today == mtryYmd && today < mtryYmd` → 수학적으로 항상 false (동시 만족 불가)
- **해결:**
```js
scwin.sbm_조회_submitdone = function(e) {
    var today = com.date.getServerDateTime().substring(0, 8); // yyyyMMdd
    var cnt = grd_아이디.getRowCount();
    for (var i = 0; i < cnt; i++) {
        var mtryYmd = dlt_loanIntSprtBizListVO.getCellData(i, "mtryYmd");
        if (today >= mtryYmd) { // 만기일 당일 포함
            grd_아이디.setCellData(i, "컬럼ID", "종료");
        }
    }
};
```
- **원칙:** YYYYMMDD 8자리 String은 사전순 비교로 날짜 대소 비교 가능

### 3. 그리드 헤더에 w2:udv_help 아이콘 추가

- **증상:** `w2:udv_help` 태그 추가해도 헤더에 아이콘 미표시
- **원인:** 기존 컬럼 `value="텍스트"` 속성 방식은 자식 컴포넌트 삽입 불가
- **해결:** `w2:header` 자식 구조로 변경

```xml
<!-- AS-IS: 속성 방식 (자식 컴포넌트 삽입 불가) -->
<w2:column displayMode="label" id="column5" inputType="text" 
    value="선사분류" width="115" style="height:28px;"/>

<!-- TO-BE: w2:header 자식 구조 -->
<w2:column displayMode="label" id="column5" inputType="text" 
    width="115" style="height:28px;">
    <w2:header>
        선사분류<w2:udv_help id="udc_help" style="width:;height:;"></w2:udv_help>
    </w2:header>
</w2:column>
```

- **팝업 텍스트 설정 위치:** `ondataload`에서 `setLabel` 호출
```js
scwin.ondataload = function() {
    udc_help.setLabel(`[중소기업확인서 발급 요건]\n- 대상: 운수 및 창고업(H) 명위 기업\n- 기준: ① 연 매출액 1,000억 원 이하 & ② 자산 총액 5,000억 원 미만\n(기준 초과 시에도 사유에 따라 향후 5년간 지위 유지 가능)`);
};
```
- **줄바꿈:** 백틱 템플릿 리터럴 내 `\n` 사용. HTML 렌더링이면 `<br>` 사용

## 핵심 학습

### WebSquare
- `com.win.message` 비동기 → 검증 분기 후 반드시 `return`
- 그리드 헤더에 컴포넌트 삽입 시 `value` 속성 방식 불가 → `w2:header` 자식 구조 필수
- `submitdone` 콜백에서 그리드 후처리(setCellData 루프) 수행
- 날짜 비교: YYYYMMDD 8자리 String 사전순 비교로 `>=`, `>` 사용 가능

### 논리 버그 유형
- `A == B && A < B` → 항상 false. 조건 재검토 필수
- 의도가 "경과 포함"이면 `>=`, "경과 후"면 `>` 명확히 구분
