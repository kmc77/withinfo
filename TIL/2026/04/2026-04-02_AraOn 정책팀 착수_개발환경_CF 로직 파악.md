## 4. CF 핵심 서비스 — `CashFlowCmptSchdService`

정책CF팀의 핵심 로직. 조건변경 시 스케줄 재산출 담당.

| 메서드 | 역할 | 핵심 로직 |
|--------|------|-----------|
| `selectCndtChgSchd` | 조건변경 스케줄 조회 | 원금유예 회차 이전 보존 → 상환방식별 재산출 |
| `selectPrinRpmtGrc` | 원금상환유예 CF 조회 | 유예 회차 이전 보존 → 변동금리 분기 |
| `selectEarlRpmt` | 조기상환 CF 조회 | 조기상환일 기준 이자 계산 → 이후 스케줄 재산출 |
| `selectMtryYmdChg` | 만기일자변경 CF 조회 | 이자계산여부 분기 → `calculateInt` 호출 |
| `selectDivdCfmtn` | 배당확정 CF 조회 | 배당확정/추가배당 분기 → 기존 row에 가산 or 추가 |
| `selectEtc` | 기타 CF 조회 | 상환일자변경 / 원금이자변경 분기 |
| `calculateInt` | 이자계산 | `intFgrsWaysCd`(이자적용방식코드)에 따라 분기 — 30/360 방식 시 일수 차이 발생 주의 |
| `generate` | 스케줄 생성 | 1회차 = 최초상환일자 강제 / 마지막 회차 만기일 맞춤 추가 |

**파라미터 공통**: `CashFlowCmptSchdReqVO` (요청) → `List<CashFlowCmptSchdVO>` (결과)
